class LongestChartParser(BottomUpProbabilisticChartParser): (source)

View In Hierarchy

A bottom-up parser for PCFG grammars that tries longer edges before shorter ones. This sorting order results in a type of best-first search strategy.
Method sort_queue Sort the given queue of Edge objects, placing the edge that should be tried first at the beginning of the queue. This method will be called after each Edge is added to the queue.
