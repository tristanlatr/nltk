class NLTKWordTokenizer(TokenizerI): (source)

View In Hierarchy

The NLTK tokenizer that has improved upon the TreebankWordTokenizer. The tokenizer is "destructive" such that the regexes applied will munge the input string to a state beyond re-construction. It is possible to apply `TreebankWordDetokenizer.detokenize` to the tokenized outputs of `NLTKDestructiveWordTokenizer.tokenize` but there's no guarantees to revert to the original string.
