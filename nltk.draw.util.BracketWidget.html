class BracketWidget(AbstractContainerWidget): (source)

View In Hierarchy

A canvas widget that places a pair of brackets around a child widget. Attributes: - ``color``: The color used to draw the brackets. - ``width``: The width of the brackets. - ``draggable``: whether the text can be dragged by the user.
