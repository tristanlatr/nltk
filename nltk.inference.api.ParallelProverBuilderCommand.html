class ParallelProverBuilderCommand(BaseProverCommand, BaseModelBuilderCommand): (source)

View In Hierarchy

This command stores both a prover and a model builder and when either prove() or build_model() is called, then both theorem tools are run in parallel. Whichever finishes first, the prover or the model builder, is the result that will be used. Because the theorem prover result is the opposite of the model builder result, we will treat self._result as meaning "proof found/no model found".
